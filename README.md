# 📊 Calculadora de Média em Python

Uma calculadora simples feita em **Python** que permite ao usuário inserir vários números e calcular a média deles.

---

## 🚀 Funcionalidades

- Inserir vários números
- Cálculo automático da média
- Exibição do resultado formatado
